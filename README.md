# PRODIGY_WD_01

## Responsive Landing Page

This project is a responsive landing page developed as part of the Prodigy InfoTech Web Development Internship.

## Features

- Fixed navigation menu visible on all pages
- Navigation menu changes background color when scrolling
- Hover effects on menu items
- Smooth scrolling navigation
- Responsive design for desktop and mobile devices
- Interactive "Explore More" button
- Multiple sections (Home, About, Services, Contact)

## Technologies Used

- HTML5
- CSS3
- JavaScript
